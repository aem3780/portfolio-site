In A Pickle: iOS Cooking App

About the project:

  • Software: Xcode, Swift, SwiftUI, Core Data
  • Skills: App development, UX/UI design
  • Duration: 6 weeks

At a Glance:

In a Pickle is for people who enjoy cooking or want to start cooking and need help organizing their grocery lists and finding recipes. The app helps people to organize their favorite recipes, meal plan, find new recipes, create shopping lists, and search for recipes.

The solution:

In a Pickle is a mobile iOS application that was designed and coded using Xcode, Swift, SwiftUI, and core data. The app consists of four screens: a list view, recipe search view, a cookbook view, and a meal plan calendar view. The recipe view loads in recipe data from a stored plist and lists the individual recipes for users to click on and follow along with. The recipes are searchable for better accesibility. These recipes can be favorited and added to the users cookbook for quicker, easier access.

The list view consists of two different lists, a grocery list and a pantry list. The grocery list allows the user to make a list of things they need to buy and check them off as they go. The pantry list allows the user to create a list of items and ingredients they already have and label the quantity of the item so they can track what they're running low on. The meal plan view allows users to organize their meals for multiple days and label them by type of meal. The list items, favorited recipes, and meal plan items are all stored in core data so that the data stays present when the app is restarted or reopened.
